# 关于 PPT 排版问题

> 排版风原则：对齐、重复、对比、亲密

## 一. 对齐

1. 目标对齐（上下外）
   - 左对齐、右对齐、顶部对齐、底端对齐
   - 水平居中、垂直居中、横向分布、纵向分布

2. 恢复性解决对齐

   - **Logo对齐**: 边框对齐
   - **人物图片对齐**: 眼神对齐

## 二. 亲密

目的：提供信息，减少混乱。

方法：调节间距，添加线条或线框。
- 当信息量多时，适当调距以及添加线条以减少信息量带来的混乱。
- 关键内容距离调节，间距要大一些。
- 当页面上的信息重复较多时，添加线框可以把相关内容统一起来以增加清晰度。
- 关键的内容应该分割开来，并保持清晰。

## 三. 对比

1. 改变标题字号，大小对比
2. 混合排字体，字体对比（字体形态过少）
3. 文章节构，相邻对比
4. 颜色对比：改变文字颜色、改变宣传板、海报等。

## 四. 重复

原则：在页面上能看到的元素都要重复。
- 如：颜色、款式、线条、形态、图片、logo。
