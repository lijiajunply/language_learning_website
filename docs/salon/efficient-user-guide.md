## 一、 认识你的电脑：核心硬件“五脏六腑”

在开始任何操作之前，了解电脑的基本构成至关重要。这就像认识一位新朋友，你需要知道它的“大脑”、“心脏”和“骨架”。

*   **CPU (中央处理器)**：电脑的“大脑”，负责处理所有指令和计算。
*   **内存 (RAM)**：电脑的“临时工作台”，用于存放正在运行的程序和数据，关机后清空。
*   **硬盘 (存储)**：电脑的“永久仓库”，用于存放操作系统、软件和所有个人文件。
*   **GPU (显卡)**：电脑的“视觉艺术家”，专门负责处理图像、视频和游戏画面。
*   **主板**：电脑的“骨架”和“神经系统”，所有其他硬件都连接在上面并进行通信。
*   **电源**：电脑的“心脏”，为所有部件提供稳定电力。

## 二、 新电脑到手，验机是第一步！

购买新电脑后，不要急于联网激活，严谨的验机流程能帮你避免买到问题机或翻新机。

### **第一步：外观检查 (全程手机录像)**
1.  **录制视频**：从开箱前开始，全程录制，确保电脑不离开摄像头范围。
2.  **外包装**：检查外箱是否有严重磕碰，封条是否完整。
3.  **机身检查**：
    *   检查机身、屏幕表面有无划痕、损伤或污渍。
    *   拿起电脑轻摇，听内部是否有零件松动异响。
    *   将电脑放平，单手开合屏幕，检查转轴阻尼是否顺滑、无异响。

### **第二步：开机前验证**
*   **运输模式**：在不插电源的情况下，直接按电源键，正常的新电脑**不应开机**。这证明电脑处于出厂运输模式。

### **第三步：开机与跳过联网**
1.  插上电源，按下开机键。
2.  **跳过联网激活**：在系统要求联网的界面，按下 `Fn + F10`，在弹出的命令窗口中输入：
    ```cmd
    start ms-cxh: localonly
    ```
    按回车，即可跳过联网步骤。
3.  **设置用户名**：建议使用英文或数字，避免中文名可能带来的软件兼容性问题。密码可按需设置。

### **第四步：硬件信息检查 (关键步骤)**
1.  **检查通电时间**：
    *   以管理员身份打开“终端”或“命令提示符”。
    *   输入命令：`powercfg /batteryreport` 并回车。
    *   命令运行后，会生成一个文件路径。复制该路径，在浏览器地址栏粘贴并访问，即可查看电池报告，其中包含**系统安装时间**和**电池首次使用时间**。新机通常只有一次记录。
    *   **注意**：通电记录可被重置，需结合其他检查。

2.  **使用专业工具深度检测**：
    *   准备一个装有“图吧工具箱”的U盘，将其拷贝到新电脑上运行。
    *   **硬件信息**：核对CPU、内存、显卡、硬盘等配置是否与购买型号一致。
    *   **硬盘检测**：使用工具箱内的“DiskInfo(64位)”工具，查看硬盘的**通电时间**（新机应小于50小时）和**总读写量**（应非常低）。
    *   **屏幕检测**：使用“坏点与漏光测试”工具，检查屏幕是否有坏点、亮点或严重漏光。
    *   **键盘检测**：使用“Keyboard Test Utility”工具，依次按下每个按键，确保全部触发正常。

### **第五步：压力测试 (“烤机”)**
此步骤测试电脑在高负荷下的稳定性和散热性能。
1.  在电脑自带的控制中心软件中，开启“独显直连”模式和最高档的“性能模式”。
2.  **CPU压力测试**：打开图吧工具箱中的“AIDA64”，点击“系统稳定性测试”（小火苗图标），只勾选“Stress FPU”，测试10分钟。期间电脑不应蓝屏、死机或黑屏。
3.  **GPU压力测试**：打开“FurMark”工具，将分辨率设置为`2560x1440`，抗锯齿(MSAA)选择`Off`，进行10分钟烤机测试。观察画面是否正常，有无花屏。

## 三、 系统简单优化设置

验机无误并激活系统后，进行一些优化设置能让电脑更好用。

1.  **关闭设备加密** (BitLocker)：对于个人用户，可关闭此功能以避免忘记密钥导致数据丢失。
2.  **修改Edge浏览器主页**：设置为常用页面，如新标签页或指定网站。
3.  **关闭系统广告**：在“设置”->“隐私”中，关闭“个性化广告”等选项。
4.  **软件下载与管理**：建议从官方渠道或可信平台下载软件。安装时注意取消勾选不必要的捆绑软件。
5.  **性能调控**：在“设置”->“系统”->“电源与睡眠”->“其他电源设置”中，根据需求选择“平衡”或“高性能”模式。
6.  **显示桌面图标**：在桌面右键“个性化”->“主题”->“桌面图标设置”，勾选需要显示的图标（如“此电脑”、“回收站”）。
7.  **关闭用户账户控制(UAC)弹窗** (谨慎操作)：在控制面板中调整UAC等级至最低，可减少安装软件时的频繁提示。
8.  **防止睡眠后无法唤醒**：更新主板、显卡驱动，或在电源设置中调整睡眠相关选项。
9.  **定期缓存清理**：可使用系统自带的“磁盘清理”工具，或使用“火绒”等安全软件的清理功能。
10. **安装安全与管理软件**：如“火绒安全”，轻量且能拦截弹窗。**图吧工具箱**可留作日后检测维护使用。
11. **配置NVIDIA控制面板**：针对游戏玩家，可在此处设置全局显卡选项，优化游戏性能。
12. **迁移社交软件文件路径**：将QQ、微信的默认文件存储位置从C盘（系统盘）更改到D盘等其他分区，防止C盘空间不足。
13. **学习常用快捷键**：如 `Win + E` (打开文件资源管理器)、`Win + Shift + S` (截图)等，极大提升效率。
14. **日常保养建议**：
    *   避免在床铺、沙发等柔软表面使用，堵塞散热口。
    *   定期用软布和压缩空气清洁键盘及散热风扇灰尘。
    *   避免电量完全耗尽再充电，随用随充。
15. **显示文件扩展名**：在文件资源管理器“查看”选项中，勾选“文件扩展名”，便于识别和修改文件类型。
16. **理解刷新率与帧数**：屏幕刷新率（如144Hz）是硬件上限，游戏内帧数（FPS）是实际输出。帧数≤刷新率时，体验最流畅。

## 四、 最后建议

*   **数据备份**：重要文件务必遵循“3-2-1”备份原则（至少3份副本，2种不同介质，1份异地备份）。
*   **保持系统更新**：定期安装Windows更新和驱动程序，以获得安全补丁和性能改进。
*   **理性安装软件**：避免安装来源不明或功能重复的软件，保持系统整洁。
